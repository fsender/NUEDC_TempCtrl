## 开发环境: ESP32C3 + Arduino + PlatformIO
****

依赖的库:

TFT_eSPI, U8g2_for_TFT_eSPI.

项目内新建名为 lib 的文件夹, 然后把库文件夹放在里面.

需要的 User_Setups 文件放在 test 文件夹下了.

替换原文件夹里面的 User_Setups 文件夹和 User_Setup_Select.h 文件即可.
